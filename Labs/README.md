# CSE215 Labs

## Overview
This folder contains hands-on lab exercises designed to reinforce concepts from lectures. Each lab folder includes instructions, starter code, and solution guidelines.

## Lab Structure
```
Labs/
├── Lab01_GettingStarted/    # Java setup & basics
├── Lab02_ClassesObjects/    # OOP fundamentals
├── Lab03_Inheritance/       # Inheritance & polymorphism
├── Lab04_Interfaces/        # Interfaces & abstraction
├── Lab05_Exceptions/        # Error handling
├── Lab06_Collections/       # Collections framework
├── Lab07_FileIO/            # File operations
└── Lab08_MiniProject/       # Integration project
```

## How to Complete Labs

1. **Read** the `README.md` in each lab folder
2. **Complete** the exercises in the provided starter files
3. **Test** your solutions using the test cases
4. **Submit** according to your instructor's guidelines

## Grading
| Component | Weight |
|-----------|--------|
| Correctness | 60% |
| Code Style | 20% |
| Comments & Documentation | 10% |
| Edge Cases | 10% |

## Tips
- Start early - don't wait until the deadline
- Test incrementally - don't write all code then test
- Ask questions in lab sessions
- Review lecture notes before attempting labs
