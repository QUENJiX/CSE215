# CSE215 Syllabus (Java & OOP)

## Course Overview
This course introduces Java programming and core OOP principles: classes, inheritance, polymorphism, interfaces, exception handling, collections, and basic concurrency.

## Learning Outcomes
- Design and implement Java classes and interfaces
- Apply OOP principles to structure programs
- Use collections and generics effectively
- Handle exceptions and perform file I/O
- Build small multi-class projects

## Weekly Topics (suggested)
1. Java basics, types, control flow
2. Classes, objects, constructors
3. Methods, encapsulation, access control
4. Inheritance and polymorphism
5. Abstract classes & interfaces
6. Exception handling & debugging
7. Collections & generics
8. File I/O and serialization
9. Unit testing with JUnit (optional)
10. Threads & concurrency (intro)
11. Design patterns (intro)
12. Project presentations

## Assessment (example breakdown)
- Assignments: 40%
- Labs: 20%
- Project: 25%
- Quizzes / Participation: 15%
